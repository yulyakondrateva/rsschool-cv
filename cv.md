## Code Example

__Task:__ You get an array of numbers, return the sum of all of the positives ones.

__Note:__ if there is nothing to sum, the sum is default to 0.

```javascript
function positiveSum(arr) {
  sum = 0;
  for(var i = 0; i < arr.length ; i++){
    if( arr[i] > 0 ){
      sum = sum + arr[i]
    }
  }    
  return sum;
}
```
